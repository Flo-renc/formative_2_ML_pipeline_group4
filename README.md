### Project Overview:
This project implements a multimodal authentication system using facial recognition and voice verification before providing product recommendations.

### Project Structure:
- `Task1_Data_Merge/` - Merge customer_social_profiles and customer_transactions datasets
- `Task2_Image_Preprocessing/` - Facial image collection, augmentation, and feature extraction
- `Task3_Audio_Processing/` - Voice sample processing and feature extraction
- `Task4_Model_Creation/` - ML models for face recognition, voice verification, product recommendation
- `Task5_System_Demonstration/` - Command-line system simulation
- `datasets/` - Raw and processed datasets
- `scripts/` - Reusable Python scripts
